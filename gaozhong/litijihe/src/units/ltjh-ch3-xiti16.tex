\xiaoti{在三面角中，如果有两个二面角是直二面角，那么它们所对的两个面角都是直角。 为什么？}

\xiaoti{直三面角内有一点 $P$，它到各面的距离分别是 $x$、$y$、$z$。用 $x$、$y$、$z$ 表示点 $P$ 到三面角顶点 $O$ 的距离。}

\xiaoti{直三面角 $O{-}XYZ$ 内有一点 $P$，$OP$ 在三面角三个面上的射影长分别是 $a$、$b$、$c$。
    求证：$OP = \sqrt{\dfrac{a^2 + b^2 + c^2}{2}}$。
}

\xiaoti{在三面角 $S{-}ABC$ 中，$\angle BSC = 90^\circ$，$\angle ASB = \angle ASC = 60^\circ$。}
\begin{xiaoxiaotis}

    \xxt{设 $SA = SB = SC$。 求证：经过三点 $A$、$B$、$C$ 的平面垂直于 $\angle BSC$ 所在的平面；}

    \xxt{求证：$SA$ 与 $\angle BSC$ 所在的平面成 $45^\circ$ 的角。}

\end{xiaoxiaotis}
