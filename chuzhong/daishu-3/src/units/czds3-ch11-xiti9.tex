\xiaoti{（我国古代问题）\footnotemark
    长方形田的面积等于 $864$ 平方步（步是我国古代的一种长度单位），长与宽的和是 $60$ 步。
    长与宽各是多少步？
}
\footnotetext{这道题选自我国南宋数学家杨辉所著《田亩比类乘除算法》（1275年）。
    原题是：“直田积八百六十四步，只云长阔共六十步，问阔及长各几步。笞曰：阔二十四步，长三十六步。”
}
