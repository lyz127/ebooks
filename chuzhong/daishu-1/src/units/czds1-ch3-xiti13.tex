\begin{xiaotis}

列出一元一次方程解下列应用题：

\xiaoti{某人买了 6 千克苹果，付出 20 元，找回 3 元 2 角。 每千克苹果的价格是多少？}

\xiaoti{用 76 厘米长的铁丝做一个长方形的教具，要使宽是 16 厘米，长应当是多少厘米？}

\xiaoti{某厂去年 10 月份生产机器 205 台， 这比前年 10 月份产量的 2 倍还多 15 台。 这个厂前年 10 月份生产机器多少台？}

\xiaoti{一个大箱子装有一种货物 36 千克。 把大箱子里的货物分装在 4 个同样大小的小箱子里，
    装满后还剩余 2 千克的货物，每个小箱子装有货物多少千克？
}

\xiaoti{某钢铁厂要用含铁量是 $58.5\%$ 的矿石炼出 3000 吨铁，需用这种旷石多少吨（精确到 10 吨）？}

\xiaoti{酒精加水稀释后，得到的一种酒精溶液的体积是原来酒精的体积的 4.5 倍。 要得到这样的酒精溶液 900 毫升，需要酒精多少毫升？}

\xiaoti{某种干饲料加水发酵后，重量可以增加 $150\%$。 为了得到 360 千克发酵饲料，需要多少千克干饲料？}

\xiaoti{地球上水面的面积约等于陆地面积的 $2\dfrac{13}{29}$ 倍，地球的表面积约等于 5.1 亿平方千米，
    求地球上水面和陆地的面积各是多少（ 精确到 0.1 亿平方千米）。
}

\xiaoti{某钢铁厂要锻造长、宽、高分别为 260 毫米、150 亳米、130 亳米的长方体毛坯，
    原料是截面积为 $130 \times 130 \; \pfhm$ 的方钢，需要截这种方钢多长（不算加工余量，下同） ？
}

\xiaoti{某机械厂要用直径为 120 毫米的圆钢锻造直径为 300 毫米，厚为 32 毫米的圆盘。应截取多长的圆钢？}

\xiaoti{某农机厂用直径为 90 毫米的圆钢锻造一种毛坯， 它是一个底面积为 $131 \times 131 \; \pfhm$，
    高为 81 毫米的长方体。 求应截取圆钢的长度（精确到1 毫米）。
}

\xiaoti{用直径为 4 厘米的圆钢，锻造一个重量是 0.62 千克的零件毛坯。如果这种钢每立方厘米的重量是 7.8 克，
    应截取圆钢多长（精确到 0.1 厘米）？
}

\xiaoti{甲、乙两人骑自行车，同时从相距 65 公里的两地相向而行，经过 2 小时相遇。
    已知甲比乙每小时多走 2.5 公里，求乙每小时走多少公里。
}

\xiaoti{甲、乙两架飞机同时从相距 750 公里的两个机场相向飞行，飞了半小时相遇。
    如果甲飞机的速度是乙飞机的速度的 $1\dfrac{1}{2}$ 倍，求这两架飞机的速度。
}

\xiaoti{甲槽有水 34 升， 乙槽有水 18 升。现在两槽同时排水，都是平均每分排出 2 升。
    多少分后， 甲槽的水是乙的水的 3 倍？
}

\xiaoti{有水田 108 公顷，旱地 54 公顷。 现在计划把一部分旱地改为水田，
    使旱地只占水田的 $20\%$。 改为水田的旱地应是多少公顷？
}

\xiaoti{甲仓存粮食 30 吨， 乙仓存粮食 40 吨。 要再往甲仓和乙仓共运去粮食 80 吨，
    使甲仓粮食数量为乙仓粮食数量的 1.5 倍。 应往甲仓和乙仓分别运去粮食多少吨？
}

\xiaoti{配制一种农药， 其中生石灰、硫磺粉和水的重量的比是 $1 : 2 : 14$。
    要配制这种农药 2550 千克，各种原料分别需要多少千克？
}

\xiaoti{一种混凝土由水、水泥、黄沙、碎石搅拌而成。 这四种原料的重量的比是 $0.7 : 1 : 2 : 4.7$。
    搅拌这种混凝土 2100 千克，各种原料分别需要多少千克？
}

\end{xiaotis}
