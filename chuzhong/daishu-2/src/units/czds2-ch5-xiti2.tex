列出一次方程组解下列应用题：

\begin{xiaotis}

\xiaoti{一只轮船顺流航行（此时航速 $=$ 船在静水中的速度 $+$ 水速）， 每小时航行 20 千米；
    逆流航行（此时航速 $=$ 船在静水中的速度 $-$ 水速）， 每小时航行 16 千米。
    求轮船在静水中的速度及水流的速度。
}

\xiaoti{某校师生 100 人到第一、第二两个车间参加劳动， 到第一车间的人数比到第二车间的人数的 2 倍少8 人。
    到两个车间参加劳动的人数各是多少？
}

\xiaoti{用白铁皮做罐头盒， 每张铁皮可裁盒身 16 个， 或裁盒底 43 个， 一个盒身和两个盒底配成一个罐头盒。
    现有 150 张白铁皮，用多少张裁盒身， 多少张裁盒底， 正好能够配成整套罐头盒？
}

\begin{enhancedline}
\xiaoti{建新村有土地 58 公顷， 计划将 10 公顷山坡地开辟为果园，其余的土地种植粮食和蔬菜，
    种粮食的土地公顷数是种蔬菜的 $4\dfrac{1}{3}$ 倍。 求计划种粮食和蔬菜的土地各是多少公顷。
}

\xiaoti{一批机器零件共 840 个。 如果甲先做 4 天， 乙加入合做，那么再做 8 天正好完成；
    如果乙先做 4 天，甲加入合做，那么再做 9 天也正好完成。 两人每天各做多少个？
}

\xiaoti{甲、乙两人从相距 18 千米的两地同时出发， 相向而行，$1\dfrac{4}{5}$ 小时相遇。
    如果甲比乙先出发 $\dfrac{2}{3}$ 小时， 那么在乙出发后 $1\dfrac{1}{2}$ 小时两人相遇。
    求两人每小时各走多少千米。
}

\xiaoti{甲、乙两人从同一地点出发， 同向而行， 甲骑自行车， 乙步行。
    如果乙先走 12 千米， 那么甲用 1 小时能追上乙；
    如果乙先走 1 小时， 那么甲只用 $\dfrac{1}{2}$ 小时就能追上乙。
    求两人的速度各是多少。
}
\end{enhancedline}

\xiaoti{某工厂前年的总产值比总支出多 500 万元。去年的总产值比总支出多 950 万元。
    已知去年的总产值比前年增加 $15\%$ ，总支出比前年减少 $10\%$， 求前年的总产值及总支出。
}

\xiaoti{某村的两块小麦地原来可以收小麦 5730 千克。 改用良种后， 这两块地可以收小麦 6240 千克，
    其中第一块地的收成增加了 $10\%$， 第二块地的收成增加了 $8\%$。 求这两块地原来各可以收小麦多少千克。
}

\xiaoti{甲、乙两工厂原计划在上个月共生产机床 360 台， 结果甲厂完成了计划的 $112\%$，
    乙厂完成了计划的 $110\%$。 两厂共生产了机床 400 台。 上个月两厂各超额生产机床多少台？
}

\xiaoti{一列火车装运一批货物， 原来每节车皮平均装 46 吨， 结果有 100 吨货物未能装进去；
    后来改进装车方法， 使每节车皮多装 4 吨， 结果把这批货装完后， 还剩下两节空车皮。
    这列火车有多少节车皮？ 这批货物有多少吨？
}

\xiaoti{食堂存煤， 如果每天用 130 千克， 按预计天数计算， 就缺少 60 千克；
    如果每天用 120 千克， 那么到期后还可剩余 60 千克。食堂存煤多少千克？ 预计用多少天？
}

\xiaoti{已知梯形的面积为 $42\pflm$， 高为 6 厘米， 它的下底比上底的 2 倍少 1 厘米。
    梯形的上底、下底各为多少厘米？
}

\xiaoti{把含盐 $5\%$ 的食盐水和含盐 $8\%$ 的食盐水混合制成含盐 $6\%$ 的食盐水600 克， 应取两种盐水各多少克？}

\xiaoti{要配制浓度为 $10\%$ 的硫酸溶液 1000 千克，已有浓度为 $60\%$ 的硫酸溶液 85 千克，
    还需浓度为 $98\%$ 的硫酸溶液和水各多少千克？
}

    \xxt[\xxtsep]{在等式 $y = kx + b$ 中， 当 $x = 0$ 时， $y = 2$ ； 当 $x = 3$ 时， $y = 3$。求 $k$， $b$ 的值。}

    \xxt{已知 \\[.5em]
        $\begin{cases} x = -1, \\ y = -1, \end{cases}$ \quad
        $\begin{cases} x = 2,  \\ y = 6,  \end{cases}$ \quad
        $\begin{cases} x = -5, \\ y = 9   \end{cases}$ \\[.5em]
        三对数值都能满足 $x^2 + y^2 + Dx +Ey + F = 0$。
        求 $D$，$E$，$F$ 的值。
    }

\end{xiaoxiaotis}
